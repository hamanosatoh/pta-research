全国PTA資料館

全国各地の学校配布文書、PTA会則、入会案内、会費徴収文書、個人情報同意書、教育委員会回答などを、単なる置き場ではなく比較可能なアーカイブとして整理する入口ページです。

How this archive works

資料館の基本設計

資料をただ並べるだけでは、研究サイトとしての価値は上がりません。必要なのは、メタデータです。自治体名、学校名、年度、文書種別、論点タグ、原資料URLを揃え、同種文書同士を横断比較できる状態にします。

推奨メタデータ：自治体名 / 学校名 / 学校種別 / 文書年度 / 文書種別 / 論点タグ / 任意加入説明の有無 / 個人情報記載 / 会費徴収方式 / 学校関与記載 / 教育委員会見解 / 原資料リンク / 注記

主要コレクション

入学時配布文書

入会案内、入会届、退会届、同意書、学校文書との混在状況を比較します。

任意加入説明不足学校文書混在

会費徴収・学校徴収金文書

口座振替依頼、団体費記載、学校名義通知、徴収依頼書などを比較します。

会費徴収公費私費分離学校関与

個人情報関連文書

名簿同意書、第三者提供説明、PTA独自収集の有無、非会員把握の構造を比較します。

個人情報第三者提供名簿

教育委員会回答・要綱・Q&A

自治体がPTAをどう位置づけ、どこまで学校関与を認めているかを比較します。

教育委員会要綱回答比較

横浜市の団体登録も、外部から活動の所在を確認できる基礎情報として配置できます。

Google Drive と接続する

GitHub Pages 側は、解説・索引・比較の場にします。原資料の本体は Google Drive に置き、このページからタグ付きで辿れるようにするのが最も運用しやすい構成です。
